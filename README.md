# Landslide-Susceptibility-Mapping-for-Nilgiris-District-using-Advanced-Machine-Learning-Methods
Six ML models namely, logistic regression, CatBoost, XGBoost, support vector machine, random forest, and C5.0 were trained using past landslide data from the region. Along with Variation-Influence-Factors (VIFs) and Tolarance charts calculations
